# EventSphere - College Event Management System

## Project Overview

The College Event Management System is a responsive website developed using HTML, CSS, and Bootstrap 5. It allows students to view upcoming events, register for events, check event schedules, browse event photos, access FAQs, and contact event coordinators.

## Features

* Responsive Navigation Bar
* Event Listings using Cards
* Event Registration Form
* Event Schedule Table
* Gallery Section
* FAQ Accordion
* Contact Form
* Registration Success Alert and Modal

## Technologies Used

* HTML5
* CSS3
* Bootstrap 5
