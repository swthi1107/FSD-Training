# Personal Portfolio Website

## Project Overview

The Personal Portfolio Website is a responsive web application developed to showcase personal and professional information in an organized and visually appealing manner. The website serves as an online portfolio that highlights educational background, technical skills, projects, achievements, and contact information.

The project is designed using HTML, CSS, and Bootstrap 5, ensuring responsiveness across different screen sizes and devices. It provides a professional platform for presenting skills and accomplishments to potential employers, recruiters, and collaborators.

---

## Features

- Responsive Navigation Bar
- Home Section with Profile Information
- About Me Section
- Skills Section with Progress Bars
- Projects Section with Project Cards
- Achievements Section
- Contact Form
- Resume Download Button
- Bootstrap Modal for Contact Confirmation
- Mobile-Friendly Design

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
